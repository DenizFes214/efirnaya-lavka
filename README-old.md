
# Эфирная лавка с ботом

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите backend:
   ```bash
   cd backend
   node server.js
   ```

   При старте backend автоматически создаются категории и услуги в MongoDB.

## Структура категорий

- Услуги
  - Массаж в технике ATT (3000 ₽)
  - Индивидуальное женское парение (4000 ₽)
- Твёрдые шампуни (пусто)
- Гидролаты (пусто)
- Масла (пусто)

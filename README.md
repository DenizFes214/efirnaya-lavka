# 🌿 Эфирная Лавка ✨

Добро пожаловать в мистический мир **Эфирной Лавки** — Telegram MiniApp для продажи эфирных масел, трав, ритуальных предметов и магических услуг.

## 🔮 Особенности

- **🌙 Магический дизайн**: Ведьмовской стиль с коричнево-зелено-золотой палитрой
- **📱 Telegram MiniApp**: Полная интеграция с Telegram WebApp API
- **🛒 Электронная торговля**: Корзина, заказы, каталог товаров
- **👑 Админ-панель**: Управление товарами и заказами для администраторов
- **🔔 Уведомления**: Автоматические уведомления в Telegram каналы
- **🎭 Тактильная обратная связь**: Вибрация при взаимодействии с интерфейсом

## 🏺 Категории товаров

- 🌿 **Эфирные масла** — Чистые масла для ароматерапии
- 🌱 **Травы и сборы** — Магические травы и целебные сборы  
- 🕯️ **Ритуальные предметы** — Свечи, амулеты, кристаллы
- 📚 **Книги и руководства** — Древняя мудрость и знания
- ✨ **Услуги** — Ритуалы, консультации, мастер-классы
- 🧴 **Натуральная косметика** — Крема и шампуни на основе трав

## 🛠️ Технологический стек

### Backend
- **Node.js** + **Express** — Серверная логика
- **SQLite** — База данных 
- **Multer** — Загрузка изображений
- **node-telegram-bot-api** — Интеграция с Telegram

### Frontend
- **Vanilla JavaScript** — Клиентская логика
- **CSS3** — Магические стили и анимации
- **Telegram WebApp API** — Нативная интеграция

### Инфраструктура
- **Docker** — Контейнеризация
- **AMVERA** — Облачное развертывание
- **SQLite** — Встроенная база данных
